Pada website ini, saya menerapkan prinsip User Experience (UX) agar pengguna merasa nyaman saat membaca dan melihat konten.

Pertama, saya mengatur tata letak konten di tengah halaman dengan lebar yang terbatas supaya teks lebih mudah dibaca dan tidak melebar ke samping. Hal ini membantu pengguna fokus pada isi utama.

Kedua, saya menggunakan hierarki visual dengan ukuran judul yang lebih besar, jarak antar elemen yang rapi, serta warna yang kontras agar informasi mudah dipahami secara berurutan.

Ketiga, saya menambahkan interaksi dan animasi ringan, seperti efek hover pada judul dan animasi mengambang pada gambar. Animasi ini dibuat halus agar tidak mengganggu, tetapi tetap membuat website terasa hidup.

Terakhir, saya menambahkan alt text pada gambar dan menggunakan struktur HTML yang jelas agar website lebih aksesibel dan ramah bagi semua pengguna.

Dengan penerapan UX ini, website menjadi lebih nyaman, menarik, dan mudah digunakan.
